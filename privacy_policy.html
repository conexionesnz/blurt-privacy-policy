1. Information We Collect

1.1 Account and Profile Information

When you create an account or profile in Blurt, we collect:

  • Authentication Data: Email address (when using Google Sign-In), user ID, authentication tokens
  • Profile Information: Username, display name, profile photo (optional), bio (optional)
  • Game Statistics: Games played, wins, streaks, XP points, player level, game mode preferences (Classic, Adult, Drinking)
  • Achievements: Unlocked badges, equipped badges, achievement progress
  • Privacy Settings: Account visibility preferences (public, friends-only, private), leaderboard consent, country/region selection

1.2 Google Sign-In Data

When you sign in with Google, we receive:

  • Google account email address
  • Google profile name
  • Google profile photo URL
  • Google user ID (for authentication)

Note: We do not have access to your Google password. Google handles authentication securely.

1.3 Apple Sign-In Data

When you sign in with Apple (iOS only), we may receive:

  • Apple ID email address (or private relay email if you choose to hide your email)
  • Your full name (if you choose to share it)
  • Apple user ID (for authentication)

1.4 Gameplay Data

During gameplay, we collect:

  • Game Sessions: Game mode selected, number of players, rounds played, game duration
  • Player Actions: Votes cast, prompts skipped, winner selections
  • Performance Stats: Win rate, favorite game modes, play frequency
  • Content Interactions: Prompt packs used, custom prompts created (if you have Creator Pass subscription)

1.5 Social and Chat Data

  • Friend Connections: Friend lists, friend requests sent/received
  • Chat Messages: In-app chat conversations with friends (stored encrypted)
  • Game Invitations: Invites sent to friends, join requests
  • Social Sharing: Victory cards shared, invite links generated

1.6 Purchase and Subscription Data

  • In-App Purchases: Product IDs purchased (Premium Unlock, Remove Ads, Prompt Packs, Drinking Challenge Packs)
  • Subscriptions: Active subscriptions (Creator Pass, All Access, Mode-specific Passes), subscription start/end dates, renewal status
  • Purchase Verification: Purchase tokens, transaction IDs (used for server-side verification only)
  • Note: Payment processing is handled by Google Play or Apple App Store. We do not have access to your credit card information.

1.7 Device and Technical Information

  • Device Data: Device model, operating system version (Android/iOS), screen resolution, device language
  • App Data: App version, installation date, last active timestamp
  • Network Data: IP address, network connection type (WiFi/mobile data)
  • Performance Data: App crashes, errors, load times, memory usage

1.8 Location Data

  • Country/Region: For leaderboard rankings and regional content (collected only if you consent)
  • Note: We do NOT collect precise GPS location. We only collect country-level data based on your selection or IP address.

1.9 Camera and Photo Data

  • Winner Photos: Photos captured during victory celebrations (stored locally on your device, only shared if you choose to share)
  • Profile Photos: Profile picture uploads (optional)
  • Note: Camera access is only requested when you use photo features. You can decline camera permissions and still use all other features.

1.10 Notification Tokens

  • FCM Token: Firebase Cloud Messaging token for push notifications (chat messages, game invites, friend requests)

1.11 Anonymous Users

If you play without signing in, we create an anonymous user ID to save your progress locally. This allows you to upgrade to a full account later while preserving your game data. Anonymous accounts have limited features (no friends, chat, or leaderboards).

4. Data Sharing and Third-Party Services

We use the following third-party services that may collect or process your data:

4.1 Firebase Services (Google)

  • Firebase Authentication: User sign-in, account management
  • Cloud Firestore: User profiles, game data, chat messages (encrypted)
  • Firebase Cloud Messaging: Push notifications
  • Firebase Analytics: App usage analytics, event tracking
  • Firebase Cloud Functions: Server-side purchase verification, data processing
  • Firebase Storage: Profile photos, user-generated content
  • Privacy Policy: https://firebase.google.com/support/privacy

4.2 Google Sign-In

  • Used for account authentication
  • Google receives: Sign-in requests, app usage (when signed in with Google)
  • Privacy Policy: https://policies.google.com/privacy

4.3 Apple Sign-In

  • Used for account authentication (iOS only)
  • Apple receives: Sign-in requests, app usage data
  • Privacy Policy: https://www.apple.com/legal/privacy/

4.4 Google AdMob (Advertising)

  • Purpose: Display ads to free users, generate revenue

  • Leaderboards: Username, photo, stats (only if you consent to leaderboard participation)
  • Shared Content: Victory cards, custom prompt packs (when you choose to share)

4.7 We DO NOT Sell Your Data

Important: We do not sell, rent, or trade your personal information to third parties for their marketing purposes.

5. Data Security

We take data security seriously and implement multiple layers of protection:

5.1 Technical Security Measures

  • Encryption in Transit: All data transmitted between your device and our servers uses TLS/SSL encryption (HTTPS)
  • Encryption at Rest: Sensitive data (chat messages, personal info) is encrypted in Firebase Firestore
  • Secure Authentication: OAuth 2.0 for Google/Apple Sign-In, secure token storage
  • Purchase Verification: Server-side validation of all purchases to prevent fraud
  • Firebase Security Rules: Database access controls to prevent unauthorized data access
  • App Check: Firebase App Check to protect backend services from abuse

5.2 Operational Security

  • Regular security audits and updates
  • Access controls limiting who can access user data
  • Secure logging practices (no sensitive data in logs)
  • Monitoring for suspicious activity and abuse

5.3 Data Breach Response

In the unlikely event of a data breach affecting your personal information, we will:

  • Notify affected users within 72 hours (GDPR requirement)
  • Notify relevant authorities as required by law
  • Provide information about what data was affected
  • Take immediate steps to secure the breach and prevent further unauthorized access

6. Data Retention and Deletion

6.1 How Long We Keep Your Data

Data Type Retention Period
Active Account Data While your account is active
Inactive Accounts 3 years after last login, then deleted
Chat Messages 90 days (automatically deleted after)
Game Session Data 2 years (for analytics and leaderboards)
Purchase Records 7 years (for tax/legal compliance)
Analytics Data 26 months (Firebase Analytics default)
Crash Logs 90 days
Deleted Account Data 30 days (grace period for account recovery)

6.2 How to Delete Your Data

Account Deletion (Full Data Removal):

  1. Open the Blurt app
  2. Go to Profile → Settings → Account
  3. Select "Delete Account"
  4. Confirm deletion (you may be asked to re-authenticate)

What Gets Deleted:

  • Profile information (username, email, photo, bio)
  • Game statistics and achievements
  • Friend connections and chat messages
  • Purchased content (prompt packs, custom content)
  • Leaderboard entries
  • Authentication tokens and user ID

What We Retain (Legal Compliance):

  • Purchase transaction records (anonymized after 30 days)
  • Aggregated analytics data (no personal identifiers)
  • Data required for ongoing legal obligations

Note: Account deletion is permanent and cannot be undone. Active subscriptions will continue until the current billing period ends, but will not auto-renew.

6.3 Partial Data Deletion

You can delete specific data without closing your account:

  • Profile Photo: Settings → Profile → Remove Photo
  • Chat History: Chat → Long press conversation → Delete
  • Leaderboard Removal: Settings → Privacy → Opt out of leaderboards

7. Your Privacy Rights

How:

  • Analytics: Settings → Privacy → Disable Analytics

7.7 Right to Withdraw Consent

What: Withdraw consent for data processing that requires consent (notifications, leaderboards).

How:

  • Notifications: Device settings or Settings → Notifications

7.8 How to Exercise Your Rights

To exercise any of these rights, contact us at:

8. Children's Privacy (COPPA Compliance)

8.1 Age Restrictions

Important: Blurt contains adult-themed content in certain game modes. The app has the following age restrictions:

  • Classic Mode: Suitable for ages 13+
  • Adult Mode: Ages 17+ only (contains mature humor)
  • Drinking Mode: Ages 21+ only (references alcohol consumption)

8.2 No Intentional Collection from Children Under 13

Blurt is not intended for children under 13 years of age. We do not knowingly collect personal information from children under 13. If you are under 13, please do not use this app or provide any information.

8.3 Parental Controls

Parents/guardians can:

  • Request deletion of a child's data by contacting blurtcomedygame@gmail.com
  • Use device-level parental controls to restrict app installation
  • Monitor in-app purchases through Google Play Family Library or Apple Family Sharing

8.4 If We Discover Data from a Child

If we become aware that we have collected personal information from a child under 13:

  • We will delete the account and all associated data immediately
  • We will notify parents (if contact information is available)
  • We will prevent future collection from that user

8.5 For Users Ages 13-17

If you are between 13-17 years old:

  • You should have parental/guardian permission to use this app
  • Adult Mode and Drinking Mode are restricted (age-verification at mode selection)
  • Parents can request data deletion or account closure at any time

9. Advertising and Monetization

9.1 Who Sees Ads

  • Free Users: See banner ads and optional rewarded video ads
  • Premium/Subscription Users: Ad-free experience
  • "Remove Ads" Purchasers: No more ads after purchase

9.2 Types of Ads

  • Banner Ads: Small ads at top/bottom of certain screens
  • Interstitial Ads: Full-screen ads between game sessions (limited frequency)
  • Rewarded Video Ads: Optional ads that grant in-game rewards (you choose to watch)

9.3 Ad Personalization

Google AdMob may show personalized ads based on:

  • Your interests and demographics (inferred by Google)
  • Your previous ad interactions
  • Apps you've used

9.4 Opt-Out of Personalized Ads

Android:

  1. Open Settings
  2. Go to Google → Ads
  3. Enable "Opt out of Ads Personalization"

iOS:

  1. Open Settings
  2. Go to Privacy → Tracking
  3. Disable "Allow Apps to Request to Track"

Note: You will still see ads, but they won't be personalized to your interests.

9.5 Ad Safety

  • We use Google AdMob, a reputable ad network with strict policies
  • Ads are screened for inappropriate content
  • If you see an inappropriate ad, please report it to blurtcomedygame@gmail.com
